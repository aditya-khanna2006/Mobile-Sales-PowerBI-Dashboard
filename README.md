# 📱 Mobile Sales Power BI Dashboard

An interactive **Power BI dashboard** developed to analyze mobile sales performance, customer purchasing behavior, and business trends. This project transforms raw sales data into meaningful insights through interactive visualizations and KPIs, helping users make data-driven decisions.

---

## 🖼️ Dashboard Preview

![Dashboard Preview](Image/Mobile Sales Dashboard.png)

---

## 📌 Project Overview

This dashboard provides a comprehensive analysis of mobile sales across different brands, cities, payment methods, and time periods. It enables users to monitor key business metrics and uncover valuable sales insights through interactive reports.

---

## 📊 Key Performance Indicators (KPIs)

- 💰 Total Sales
- 📦 Total Quantity Sold
- 🛒 Total Transactions
- 💵 Average Selling Price

---

## ✨ Dashboard Features

- Interactive slicers for dynamic filtering
- Monthly sales trend analysis
- Brand-wise sales comparison
- City-wise sales performance
- Customer rating analysis
- Payment method distribution
- Clean and user-friendly dashboard design

---

## 🛠️ Tools & Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel
- Data Visualization

---

## 📂 Dataset

The dataset contains mobile sales transaction details, including:

- Mobile Brand
- Mobile Model
- Sales Amount
- Quantity Sold
- Customer Ratings
- Payment Method
- City
- Transaction Date

> **Note:** This dataset is used for educational and portfolio purposes only.

---

## 📈 Key Insights

- Identified top-performing mobile brands based on sales.
- Analyzed monthly sales trends to understand business performance.
- Compared sales across different cities.
- Evaluated customer ratings and purchasing behavior.
- Examined preferred payment methods used by customers.
- Tracked overall business performance using KPI cards.

## 🚀 How to Use

1. Download or clone this repository.
2. Open the `.pbix` file using **Microsoft Power BI Desktop**.
3. Refresh the data if required.
4. Explore the dashboard using the interactive filters and slicers.
